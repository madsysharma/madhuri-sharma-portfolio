Mandala Flow State

Research Project (Healthcare Innovation and Design Lab, Santa Clara University)

  • Overview: this application combines Virtual Reality with neurofeedback to boost immersion for mindfulness practices.
  • How it works:
    • The VR application (developed on Unity) is a generative 3D Mandala and soundscape to bring the user to a focused, relaxed state.
    • An oscillating cloud in the middle of the screen helps to guide the user's breathing.
    • Using an electroencephalography (EEG) powered device, raw EEG data is collected from two (frontal) electrodes. A Python script preprocesses the data to clean it and eliminate any noise.
    • The power of brain activity in the alpha frequency band (which reflects a relaxed and alert state) is extracted from the cleaned data and is used to compute a feature indicating the user's level of focus and relaxation.
    • The computed feature is then wirelessly transmitted to the VR headset where it is represented in the form of the aforementioned cloud. With such a continuous feedback loop over the span of seven minutes, the user is guided to a focused and relaxed state owing to the changing opacity and size of the cloud (which increase with an increase in alpha frequency activity).
  • My roles and responsibilities:
    • Data extraction and preprocessing
    • Created a new feature where the user can input the number of sessions using a new keypad scene:
      • Based on the number of sessions completed, the cloud's starting opacity and diameter is adjusted.
  • The project overview, written by my mentor, Dr. Julia Scott can be found here.

Mar 2024 - Jun 2025

Fabled Forest: An Immersive Experience for Stress Management Using Biofeedback and Gesture Control

Research Project (Healthcare Innovation and Design Lab, Santa Clara University)

  • Overview: an original virtual reality (VR) application that combines open-world game design, gesture-based user interaction, and biofeedback to modulate the user's physiological and psychological states in an interactive and immersive experience.
  • Design:
    • Users explore a mountainous rainforest through natural gestures, free from the constraints of traditional controllers.
    • The user enters minigames by porting through entryways along the mountain path.
    • Minigames include the relaxing activity of “Skipping Stones,” where users gently toss pebbles across a pond to create ripples accompanied by calming sound effects.
    • At the conclusion of the experience atop the mountain, “Mandala Flow State” guides users in slow breathing patterns.
    • The various factors of the environmental weather (lighting, wind, etc) represent the change in Heart Rate Variability (HRV), while visual effects are modulated by the alpha metric, providing feedback on the transition to a focused relaxation state.
    • As each minigame is completed, a layer of the mandala is acquired, which progressively creates the traditional nine-layer architecture.
  • Presented at the MeaningfulXR conference of 2025 (held at UC Davis)
  • Code and poster available on approved request only (please contact Dr. Julia Scott: jscott1@scu.edu)
